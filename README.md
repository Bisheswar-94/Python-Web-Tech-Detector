# Python-Web-Tech-Detector
Python Web Tech Detector is a cybersecurity reconnaissance tool built with Python that analyzes websites through HTTP headers. It detects web servers, backend technologies, frameworks, CDN/WAF protection, and important security headers to help users understand website technologies and security configurations.
